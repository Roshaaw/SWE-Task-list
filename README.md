### **Project Overview** 
**SWE Task List App**  is a full-stack Task List web application built using Flask and MongoDB Atlas.
The application is deployed on AWS EC2 and automatically updates using GitHub Actions (CI/CD)

---

## The public EC2 URL
[Link Here](http://ec2-54-153-24-43.us-west-1.compute.amazonaws.com:8080/)

---

### Core Features  
-  **Add new tasks**
-  **Prevent empty tasks**
-  **Mark tasks as complete**
-  **Delete tasks**
-  **Automatic timestamps for each task** 
-  **Deployed on AWS EC2**

---

## Technologies Used 

- **Frontend:** HTML, CSS    
- **Backend:** Python, Flask 
- **Database:** MongoDB Atlas
- **Others:** AWS EC2, GitHub Actions, Git
